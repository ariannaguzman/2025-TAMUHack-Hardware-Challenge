# Sweep

Sweeps the shaft of an RC [servo motor](http://en.wikipedia.org/wiki/Servo_motor#RC_servos) back and forth across 180 degrees.

## Hardware Required

* Arduino Board
* 3 Servo Motor
* Jumper Wires
* LED Lights/LED Diodes
* 100 Ohm Resistance

## Circuit

LED can be replaced with a light strip or multiple LEDs in parallel.
